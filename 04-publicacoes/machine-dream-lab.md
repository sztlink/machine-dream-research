# Machine Dream Lab

## Função

O **Machine Dream Lab** é o research log público do projeto no `news.szt.link`. Ele traduz o trabalho do ateliê para uma forma pública, sanitizada, legível e citável.

Não substitui o corpus bruto.
Não é o site canônico final do artista.
É a membrana pública da pesquisa em andamento.

## Primeira peça pública

Título:

```txt
Dreaming não é apenas memória
```

Tese:

```txt
O Machine Dream não compete com dreaming como feature técnica. Ele desloca o termo para uma prática de elaboração: memória, ausência, imagem, intimidade e futuro.
```

## Próxima peça sugerida

coletar → omitir → deformar → dividir → imaginar → devolver → editar → preservar → publicar

Objetivo da peça:

## Relação com este pacote

Este pacote `machine-dream-research` deve funcionar como base metodológica do Lab. O Lab publica fragmentos narrativos; este pacote preserva a estrutura de pesquisa.

## Critério editorial

Cada publicação do Lab deve responder:

1. Qual gesto humano originou o texto?
2. O que pode ser dito publicamente sem expor o bruto?
3. Qual conceito fica mais claro depois da publicação?
4. O texto ajuda o mestrado ou apenas reage ao mercado?
5. O corpo de Felipe aceita esta exposição?
