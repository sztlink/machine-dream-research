# Fac-símiles

Esta pasta receberá apenas fac-símiles sanitizados.

## O que é um fac-símile aqui

Um fac-símile é uma amostra editada do Machine Dream que preserva a estrutura, o ritmo e a lógica metodológica do sonho sem expor o corpus bruto.

Ele não é transcrição integral.
Ele não é vazamento controlado.
Ele é uma forma pública derivada.

## Checklist antes de adicionar um fac-símile

- [ ] Remover nomes sensíveis.
- [ ] Remover paths internos.
- [ ] Remover comandos acionáveis.
- [ ] Remover URLs privadas.
- [ ] Remover dados de saúde, família, clientes, financeiro e contratos.
- [ ] Remover credenciais ou qualquer material que pareça segredo.
- [ ] Indicar claramente que houve edição.
- [ ] Preservar data, estrutura e função metodológica.
- [ ] Passar pelo teste corporal de Felipe.

## Formato sugerido

```txt
03-facsimiles/
  2026-03-31-facsimile.md
  2026-03-31-redaction-notes.md
```

Cada fac-símile deve vir acompanhado de notas de edição.
