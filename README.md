# Machine Dream Research

**Sonho da Máquina / Machine Dream** é uma pesquisa artística baseada em prática sobre memória, sonho, arquivo e prospecção em um implante cognitivo pessoal.

Este repositório organiza a camada pública e citável do projeto. Ele não contém o corpus bruto do szt.link, nem prompts completos, nem dados íntimos. Contém protocolo, política de arquivo, vocabulário, bibliografia, claims e fac-símiles sanitizados.

## Tese curta

> Não é *dreaming* como feature; é uma partitura pública para elaborar memória, revelar ausências e simular futuros de um implante cognitivo.

## Origem

O Machine Dream antecede a discussão pública recente sobre *managed agents dreaming*. Sua primeira versão foi concebida e executada em **30 de março de 2026**, após uma etapa de pesquisa sobre sonho, memória, agentes e replay prospectivo.

A formulação inicial partiu de uma pergunta prática:

> O que aconteceria se o szt.link sonhasse enquanto Felipe dorme?

A primeira arquitetura usou agentes paralelos para percorrer o workspace, produzir checkpoints e sintetizar um relatório noturno. A inspiração declarada era a neurociência do sonho, especialmente a noção de **replay prospectivo** associada ao hipocampo.

Depois da primeira execução, o método foi lapidado: deixou de ser uma auditoria operacional para se aproximar de uma prática de elaboração simbólica, incorporando corpo, ausência, memória, phármakon e arquivo.
